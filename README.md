# NSV Infra Engineering Projects Website

Static production-ready website deliverable for NSV Infra Engineering Projects.

## Pages

- `index.html`
- `about.html`
- `services.html`
- `projects.html`
- `quotation.html`
- `contact.html`

## Editable Content

- Company information, WhatsApp number, Google Form URL, map embed URL, services, projects, and testimonials live in `assets/js/site-data.js`.
- Generated site imagery lives in `assets/images/`.
- Styling lives in `assets/css/styles.css`.

## Integration Notes

- Replace `YOUR_GOOGLE_FORM_URL_HERE` in `assets/js/site-data.js` with the published Google Form URL.
- Replace `91 9177210909` and `+91 9177210909 ` with the real WhatsApp and phone numbers.
- Replace social media placeholder links in `assets/js/main.js` when official profiles are available.

Open `index.html` in a browser to preview the site.
